🔐 A beginner-friendly Python implementation of the Caesar Cipher that encrypts and decrypts messages using a customizable shift value.
## ✨ Features

- Encrypts secret messages using the Caesar Cipher technique.
- Decrypts encrypted messages back to their original text.
- Allows the user to choose the shift value.
- Supports multiple encrypt/decrypt operations in a single session.
- Uses functions to organize the encryption and decryption logic.
- Beginner-friendly Python project demonstrating loops, strings, lists, functions, modules, and conditional statements.
